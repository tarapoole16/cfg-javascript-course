# cfg-javascript-course
This repository contains my practice work and code along exercises from the Code First Girls JavaScript Coding Kickstarter course. The course introduced me to the fundamentals of JavaScript and core programming concenpts such as variables, data types, functions, arrays and loops.

I completed the eight modules in this kickstarter course which included:
- Module 1. Hello, World!
- Module 2. Data Types & Variables
- Module 3. Comparisons
- Module 4. Loops & Functions
- Module 5. Web Development
- Module 6. Web Development 2
- Module 7. Recap
- Module 8. Reflection
